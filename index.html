My First Heading

cd : Change the directory

ls : lists the files, their changes

git clone 'url': Helps us copy the project on the github to our computer

git status : Helps us know the status of the project whether it is modified or not, or not. or a new untracked file is created

git add . : Helps us to add and save all the new and modified files at once . signifies all

git add filename : Helps us to add and save a particular file

git commit -m "message" -m "some description": Helps us to add a message and a description of the changes we have done

git push: Helps us finally make the changes live in the remote repository

Inorder to push you to have ssh key which links your computer with the remote repository view the process from any article or video
